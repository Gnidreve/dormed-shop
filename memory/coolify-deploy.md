Coolify läuft **nicht mehr** auf dem Host. Der neue Host hat kein Coolify.

**Why:** Umzug auf neuen Host ohne Coolify.

**How to apply:** Nie nach Coolify-Deploy fragen. Deployment passiert manuell — Linuz pusht selbst per git oder führt den Deploy-Prozess manuell aus. Kein automatisches Deploy-Angebot.
